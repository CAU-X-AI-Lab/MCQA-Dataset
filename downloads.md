# Downloads

The GitHub repository contains documentation, statistics, scripts, and small samples.
The full dataset should be downloaded from GitHub Releases.

After creating a GitHub Release, upload the following archive files:

| Dataset | Archive | Files | Size |
|---|---|---:|---:|
| CM | `mcqa-cm.zip` | 63,980 | 66.97 MB |
| FB | `mcqa-fb.zip` | 182,133 | 162.87 MB |
| UD | `mcqa-ud.zip` | 26,697 | 95.54 MB |

Suggested release tag:

```text
v1.0.0
```

Suggested GitHub Release asset URLs after upload:

```text
https://github.com/CAU-X-AI-Lab/MCQA-Dataset/releases/download/v1.0.0/mcqa-cm.zip
https://github.com/CAU-X-AI-Lab/MCQA-Dataset/releases/download/v1.0.0/mcqa-fb.zip
https://github.com/CAU-X-AI-Lab/MCQA-Dataset/releases/download/v1.0.0/mcqa-ud.zip
```
